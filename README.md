# World of Physics

Interactive 2D physics sandbox for learning Newtonian mechanics and electromagnetism.

## Development

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open: http://localhost:3000

## Deploy to GitHub Pages

This repo includes a GitHub Actions workflow that builds a static export and deploys `out/` to GitHub Pages.
